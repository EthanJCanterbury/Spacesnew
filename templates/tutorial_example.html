Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page and consists of a series of elements that tell the browser how to display the content.

Key Concept: HTML uses "markup" to annotate text, images, and other content for display in a web browser. HTML elements are represented by tags.

Basic HTML Document Structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first web page.</p>
  </body>
</html>

Let's break down this structure:

  • <!DOCTYPE html> - Declares the document type and version of HTML
  • <html> - The root element that contains all other HTML elements
  • <head> - Contains meta-information about the document that isn't displayed
  • <meta> - Provides metadata about the HTML document
  • <title> - Specifies the title of the document (shown in browser tab)
  • <body> - Contains all the visible content of the webpage
  • <h1> - Defines a large heading
  • <p> - Defines a paragraph

HTML Elements in Detail

HTML elements are the building blocks of web pages. Each element serves a specific purpose and helps structure your content in a meaningful way.

Common HTML Elements:

<body>
  <h1>Main Heading</h1>
  <h2>Subheading</h2>
  
  <p>This is a paragraph with <strong>bold text</strong> and <em>italic text</em>.</p>
  
  <a href="https://example.com">This is a link</a>
  
  <img src="image.jpg" alt="Description of image">
  
  <ul>
    <li>Unordered list item 1</li>
    <li>Unordered list item 2</li>
  </ul>
  
  <ol>
    <li>Ordered list item 1</li>
    <li>Ordered list item 2</li>
  </ol>
  
  <div>A division or section in a document</div>
</body>

Understanding these key HTML elements:

  • Headings (<h1> to <h6>) - Used for titles and subtitles, with h1 being the largest and h6 the smallest. They help structure your document and are important for SEO and accessibility.
  • Paragraphs (<p>) - Define blocks of text. Browsers automatically add margin space before and after each paragraph.
  • Text formatting (<strong>, <em>) - Used to emphasize text. <strong> indicates strong importance (typically bold), while <em> adds emphasis (typically italic).
  • Links (<a>) - Create hyperlinks to other pages or resources. The href attribute specifies the destination URL.
  • Images (<img>) - Embed images in your webpage. The src attribute points to the image file, while alt provides alternative text for accessibility.
  • Lists (<ul>, <ol>, <li>) - Create unordered (bulleted) lists with <ul> or ordered (numbered) lists with <ol>. Each list item is defined with <li>.
  • Divisions (<div>) - Generic container elements used to group content for styling and positioning.

Introduction to CSS

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Key Concept: CSS separates the content (HTML) from its presentation, allowing you to control the layout, colors, fonts, and more across multiple web pages from a single file.

CSS Syntax:

selector {
  property: value;
  another-property: another-value;
}

There are three ways to include CSS in your HTML document:

  1. Inline CSS - Using the style attribute within HTML elements
  2. Internal CSS - Using the <style> element in the <head> section
  3. External CSS - Using an external CSS file linked with the <link> element

External CSS (Recommended):

<!-- In the HTML head section -->
<head>
  <link rel="stylesheet" href="styles.css">
</head>

/* In styles.css */
body {
  font-family: Arial, sans-serif;
  background-color: #f4f4f4;
  margin: 0;
  padding: 20px;
}

h1 {
  color: #4e54c8;
  text-align: center;
}

p {
  line-height: 1.6;
  color: #333;
  margin-bottom: 16px;
}

Best Practice: Always prefer external CSS files for larger projects as they promote separation of concerns, reusability, and easier maintenance.

CSS Selectors

CSS selectors are patterns used to select and style HTML elements. Understanding selectors is crucial for effective styling.

Common CSS Selectors:

/* Element Selector */
p {
  color: blue;
}

/* ID Selector */
#header {
  background-color: black;
  color: white;
}

/* Class Selector */
.highlight {
  background-color: yellow;
}

/* Attribute Selector */
input[type="text"] {
  border: 1px solid gray;
}

/* Descendant Selector */
article p {
  font-style: italic;
}

/* Pseudo-class Selector */
a:hover {
  text-decoration: underline;
}

Types of CSS selectors explained:

  • Element Selector - Selects all elements of the specified type (e.g., p, h1, div)
  • ID Selector (#) - Selects an element with a specific ID. IDs must be unique within a page.
  • Class Selector (.) - Selects elements with a specific class attribute. Multiple elements can share the same class.
  • Attribute Selector - Selects elements based on their attributes.
  • Descendant Selector - Selects elements that are descendants of a specified element.
  • Pseudo-class - Selects elements based on a specific state (like hover, focus, active).

Understanding Specificity: When multiple selectors apply to the same element, CSS uses specificity to determine which styles to apply. Generally, ID selectors have higher specificity than class selectors, which have higher specificity than element selectors.

CSS Box Model

The CSS Box Model is a fundamental concept that describes how elements are rendered on a web page. Every HTML element is treated as a rectangular box with content, padding, border, and margin areas.

Box Model Example:

.box {
  width: 300px;
  height: 200px;
  padding: 20px;
  border: 5px solid #333;
  margin: 30px;
  background-color: #f0f0f0;
}

Box Model Components:

  • Content - The actual content of the box where text and images appear
  • Padding - The space between the content and the border
  • Border - A line that surrounds the padding (if specified)
  • Margin - The space outside the border, separating the element from other elements

By default, the width and height properties define the size of the content area only. This means that when you add padding, border, or margin, the total dimensions of the element will be larger than the specified width and height.

Changing the Box Model:

* {
  box-sizing: border-box;
}

The box-sizing: border-box property changes the box model so that padding and border are included in the element's width and height, making layout calculations much easier.

CSS Layout Techniques

CSS offers several powerful techniques for creating layouts. Understanding these will help you create responsive and well-structured web pages.

Flexbox Layout:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.item {
  flex: 1;
  margin: 10px;
  padding: 20px;
  background-color: #e9e9e9;
}

CSS Grid Layout:

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 20px;
}

.grid-item {
  background-color: #e9e9e9;
  padding: 20px;
  text-align: center;
}

Comparing Layout Methods:

  • Flexbox - Ideal for one-dimensional layouts (rows OR columns). Great for alignment, spacing, and ordering elements.
  • CSS Grid - Perfect for two-dimensional layouts (rows AND columns). Excellent for complex grid-based designs.
  • Floats - An older technique, now mostly used for specific cases like wrapping text around images.
  • Positioning - Using position properties (relative, absolute, fixed) for specific layout requirements.

Responsive Design Tip: Modern layouts should be responsive, adapting to different screen sizes. Use techniques like media queries, flexible units (%, rem, em), and mobile-first design principles.

Introduction to JavaScript

JavaScript is a programming language that enables interactive web pages. It allows you to create dynamic content, control multimedia, animate images, and pretty much everything else.

Key Concept: JavaScript is a client-side scripting language that runs in the browser. It can respond to events, manipulate HTML content, validate forms, create cookies, and much more.

How to Include JavaScript:

<!-- At the end of the body for better performance -->
<body>
  <!-- Your HTML content -->
  
  <script>
    // Internal JavaScript
    console.log('Hello, World!');
  </script>
  
  <!-- External JavaScript -->
  <script src="script.js"></script>
</body>

JavaScript Basics:

// Variables
let name = 'John';
const age = 30;

// Functions
function greet(personName) {
  return 'Hello, ' + personName + '!';
}

// Calling a function
let greeting = greet(name);
console.log(greeting);  // Output: Hello, John!

// Conditional statements
if (age >= 18) {
  console.log('Adult');
} else {
  console.log('Minor');
}

// Arrays
let fruits = ['Apple', 'Banana', 'Cherry'];
console.log(fruits[0]);  // Output: Apple

// Loops
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

JavaScript is an essential part of modern web development and is the foundation for many frameworks and libraries like React, Angular, and Vue.js.

JavaScript DOM Manipulation

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content.

Common DOM Operations:

// Selecting elements
const heading = document.getElementById('main-heading');
const paragraphs = document.getElementsByTagName('p');
const highlighted = document.getElementsByClassName('highlight');
const firstParagraph = document.querySelector('p');
const allButtons = document.querySelectorAll('button');

// Modifying content
heading.textContent = 'New Heading';
heading.innerHTML = 'New <span>Heading</span>';

// Changing styles
heading.style.color = 'blue';
heading.style.fontSize = '24px';

// Adding/removing classes
heading.classList.add('active');
heading.classList.remove('hidden');
heading.classList.toggle('highlighted');

// Creating new elements
const newParagraph = document.createElement('p');
newParagraph.textContent = 'This is a new paragraph.';
document.body.appendChild(newParagraph);

// Removing elements
const oldElement = document.getElementById('outdated');
oldElement.parentNode.removeChild(oldElement);

Working with Web APIs

Web APIs (Application Programming Interfaces) allow your JavaScript code to interact with external services or access browser functionality. One common use is fetching data from servers.

Using the Fetch API:

// Basic fetch request
fetch('https://api.example.com/data')
  .then(response => {
    if (!response.ok) {
      throw new Error('Network response was not ok');
    }
    return response.json();
  })
  .then(data => {
    console.log(data);
    displayData(data);
  })
  .catch(error => {
    console.error('There was a problem with the fetch operation:', error);
  });

// Function to display the data
function displayData(data) {
  const container = document.getElementById('data-container');
  
  data.forEach(item => {
    const element = document.createElement('div');
    element.classList.add('data-item');
    element.textContent = item.name;
    container.appendChild(element);
  });
}

Using async/await (Modern Approach):

async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    
    if (!response.ok) {
      throw new Error('Network response was not ok');
    }
    
    const data = await response.json();
    console.log(data);
    displayData(data);
  } catch (error) {
    console.error('There was a problem with the fetch operation:', error);
  }
}

// Call the function
fetchData();

Common Web APIs:

  • Fetch API - For making HTTP requests
  • Storage API (localStorage, sessionStorage) - For storing data in the browser
  • Geolocation API - For accessing the user's location
  • Canvas API - For drawing graphics
  • Web Audio API - For sound processing and synthesis
