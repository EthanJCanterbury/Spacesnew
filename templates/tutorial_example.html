Getting Started with HTML

HTML (HyperText Markup Language) is the backbone of any website. It defines the structure and content of your web page.

Basic HTML Structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Website</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first web page.</p>
  </body>
</html>

Tip: HTML documents always start with a doctype declaration, followed by the html element which contains head and body sections.

Adding More Content

Let's add more elements to make your page more interesting and informative.

Common HTML Elements:

  • <h1> to <h6> - Headings (h1 is largest, h6 is smallest)
  • <p> - Paragraph
  • <a> - Links to other pages
  • <img> - Images
  • <ul> and <ol> - Unordered and ordered lists
  • <div> - Container for grouping elements

Try adding different headings, paragraphs, and links to your page!

Introduction to CSS

CSS (Cascading Style Sheets) is used to style your HTML elements and make your website look attractive.

Basic CSS Syntax:

selector {
  property: value;
  another-property: another-value;
}

Example CSS:

body {
  font-family: Arial, sans-serif;
  background-color: #f4f4f4;
  margin: 0;
  padding: 20px;
}

h1 {
  color: #4e54c8;
  text-align: center;
}

p {
  line-height: 1.6;
  color: #333;
}

Note: CSS can be added in three ways: inline, internal (using style tags), or external (using a separate CSS file).

Working with JavaScript

JavaScript allows you to add interactivity to your website.

Simple JavaScript Example:

<button id="myButton">Click Me!</button>

<script>
  document.getElementById('myButton').addEventListener('click', function() {
    alert('Button was clicked!');
  });
</script>

JavaScript enables you to:

  • Respond to user actions (clicks, mouse movements, etc.)
  • Modify the HTML content dynamically
  • Validate forms before submission
  • Create animations and visual effects
